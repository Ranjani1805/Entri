  ## Module End Project

-**Module End Project – `Ranjani_MySQL Capstone Project  E-Commerce Customer Churn Analysis`**

**Objective:** Analyse an e-commerce dataset to identify **customer churn patterns** and provide actionable business insights.  

**Key Highlights:**
- **Data Exploration & Cleaning:** Inspecting tables, handling missing data, and preparing the dataset for analysis.  
- **Customer Behaviour Analysis:** Understanding patterns based on transactions, tenure, complaints, and engagement metrics.  
- **Churn Identification:** Classifying **churned vs active customers** using SQL queries, aggregate functions, and CASE statements.  
- **Insights & Reporting:** Generating summaries, trends, and actionable insights to support business decisions in customer retention strategies.

**Deliverables:** SQL scripts and a detailed report documenting methodology, queries, results, and insights.
