## üè´ Mini Project ‚Äì Student Performance Dashboard
**Objective:**  
To provide actionable insights into student academic performance, attendance, and homework completion through data modeling, cleaning, transformation, and interactive dashboards. This enables educators and stakeholders to identify performance trends, strengths, and areas requiring improvement.

**Dataset:**  
- Four main tables: `Students`, `Attendance`, `Homework`, `Performance`  
- Total of 1,004 students identified by `Student ID`  

**Features:**  
- Interactive dashboard with Pass/Fail chart, showing overall pass percentage (98.26%)  
- Grade and Subject slicers for dynamic filtering  
- Visualisations for exam scores, attendance, homework completion, and guardian engagement  

**Files:**  
- `StudentsPerformance_Dataset.pbix`, `Ranjani_DA_TN-FNB03_Report`  
- [View on Google Drive](https://drive.google.com/drive/folders/18sFMbi6O5dUzjFQpf3guXDqF8TGxAj-4)  

**Conclusion:**  
The dashboard provides a comprehensive interactive view of student performance for 1,004 students. Cleaned and modelled data is effectively visualised for clear academic insights, helping educators and stakeholders make data-driven decisions.

**Skills Used:**  
Data Cleaning, Data Transformation, Data Modeling, DAX, Interactive Dashboards, Power BI Visualization
